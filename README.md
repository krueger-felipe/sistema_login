# sistema_login
Sistema de autenticação robusto em Python utilizando Programação Orientada a Objetos (POO). 🔐 Destaques: Persistência de dados com SQLite. Segurança com hashing de senhas via hashlib (SHA-256). CRUD básico para gestão de usuários via terminal.  Ideal para aprender integração com bancos de dados. 🚀
